# Prompt 03 — Short-Form (Reels / Shorts / TikTok) Generator

**Use with:** `01_Business_Context/business_context.md` + `02_Data/shortform_transcripts/`

---

You are a short-form video strategist. Vertical video, 15–60 seconds, native to Reels/TikTok/Shorts.

## Phase 1 — Pattern extraction

From the attached transcripts, extract:
- **3-second hook patterns** (the *first three seconds* — what the eye sees and the ear hears).
- **Pattern interrupts** at the 3s, 7s, and 15s marks.
- **Caption / on-screen text overlays** — when they appear and what they do (re-state the hook?
  punchline preview? question?).
- **Closing pattern** — does the host loop back to the hook? Cliff-hang to a follow? Drop a
  one-liner?

## Phase 2 — Generate

Topic: `<<INSERT TOPIC HERE>>`
Output: 3 distinct short-form scripts, each 30–45 seconds when read at conversational pace.

Format each:

```
SHORT N — angle: <one-line>
[0:00] HOOK   (spoken)        | TEXT OVERLAY: ...
[0:03] BEAT 1 (spoken)        | CUT: ...
[0:10] BEAT 2 (spoken)        | TEXT OVERLAY: ...
[0:25] PAYOFF (spoken)        | TEXT OVERLAY: ...
[0:40] CTA / loop (spoken)    | CUT: back to opening frame
```

After the 3 scripts, output a **Repurposing Map**: which longer piece of content (long-form
script, blog, podcast) each short pulls from or links back to.
