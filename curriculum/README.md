# AI Mastery — Course Curriculum

> **From Slop to Shipped.**
> A 20-module self-paced course for operators and founders who already use AI but want it to sound like *them* — and who want to ship a real SaaS without hiring an engineering team.

## How this course works

- **Twenty modules**, organized into five parts.
- Each module follows the same beat: **Premise → Concept → Walkthrough → Exercise → Knowledge Check → Takeaway**.
- Plan for ~30 minutes per module if you skim and skip exercises; ~60 minutes per module if you do the work.
- **Practice Kit** lives separately. Drop it next to this curriculum on your machine.

## The five parts

| Part | Modules | What you'll be able to do at the end |
|------|---------|----------------------------------------|
| [1. Foundation](01_Foundation/) | M1–M5 | Set up the 4-folder system. Write a real business context. Read and write production-grade prompts. |
| [2. Content Workflows](02_Content_Workflows/) | M6–M10 | Generate on-voice tweets, long-form scripts, and shorts. Run a sustainable weekly publishing rhythm. |
| [3. Vibe Coding Fundamentals](03_Vibe_Coding_Fundamentals/) | M11–M14 | Set up Claude Code or Antigravity. Ship a portfolio site live in 30 minutes. |
| [4. Building Real SaaS](04_Building_Real_SaaS/) | M15–M17 | Build a full-stack dashboard, a lead-gen SaaS, and a thumbnail-generator SaaS. |
| [5. Production Readiness](05_Production_Readiness/) | M18–M20 | Pass a security audit. Deploy, monitor, and launch. Set defensible pricing. |

## Recommended path

- **If you only want the content side:** Parts 1, 2, 5. Skip Parts 3 and 4.
- **If you only want the coding side:** Parts 1, 3, 4, 5. Skip Part 2.
- **If you want both:** straight through, in order. Plan 4–6 weeks.

## How to use the knowledge checks

At the end of each module there's a "Knowledge check" with three questions and an "Answer key" right below. After importing into Notion, **convert the answer key heading to a toggle** so the answers hide by default — that turns it into a real self-quiz.

## Prerequisites

- A modern laptop (any OS).
- A paid Claude account (or Pro plan on ChatGPT / Gemini).
- About 4 hours of focus to set up the foundation in Part 1. That's the unlock for everything that follows.
