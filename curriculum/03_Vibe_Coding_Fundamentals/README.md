# Part 3 — Vibe coding fundamentals

> The four modules that turn you from "I can't code" into "I shipped a real website."

## Why this part exists

Vibe coding is the practice of building real software primarily by *prompting* an AI agent rather than typing code by hand. To do it well, you need a working mental model of the parts (M12), a clean environment (M11), a development loop (M13), and one finished project to anchor the muscle (M14).

## Modules

| # | Lesson | Time |
|---|--------|------|
| [M11](M11_Setup.md) | Setup: Claude Code or Antigravity | 30 min |
| [M12](M12_How_web_apps_work.md) | How web apps actually work | 60 min |
| [M13](M13_Development_loop.md) | The vibe-coding development loop | 45 min |
| [M14](M14_Project1_portfolio.md) | Project 1 — Portfolio site (live in 30 min) | 60–120 min |

**Total: ~3.5 hours** (plus the build itself; the portfolio site can deploy in 30 min if you don't tune the design endlessly).

## Outcome

A live personal website at your own subdomain (`yourname.netlify.app` or a custom domain), Lighthouse scores ≥95, dark-mode toggle that persists, and the muscle to do it again for the bigger projects in Part 4.
